# prototype
Создаёт объект Person с именем и возрастом.

С помощью метода Clone() создаёт его копию.

Меняет имя клона, чтобы показать, что это отдельный объект.

Выводит данные оригинала и клона на экран.

Паттерн Prototype позволяет создавать новые объекты, копируя существующие, вместо создания с нуля.
